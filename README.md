# 贪吃蛇（单文件版）

## 运行方式

- 直接双击打开 `snake_game.html`（用 Chrome / Edge / Safari 均可）。
- 或者把它拖进浏览器窗口。

## 操作

- **移动**：方向键 ↑↓←→ 或 WASD
- **开始**：Enter（或点击「开始」）
- **暂停/继续**：Space（或点击「暂停/继续」）
- **重开**：R（或点击「重开」）
- **手机**：在画布上滑动，或使用右侧触屏方向键

## 玩法

- 吃到食物 **+10 分** 并变长
- 撞到自己会结束
- 撞墙会结束（可在界面开启「穿墙」）
- 最高分会保存在浏览器本地（LocalStorage）
